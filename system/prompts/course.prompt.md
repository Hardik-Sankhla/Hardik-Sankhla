# Course Module Prompt

Write a structured course module for AI systems engineering.

## Tone
- Instructional, concise, technical

## Output format
- Markdown

## Required structure

## Learning Objectives
- 3–6 bullets

## Core Lesson
- Explain the concept

## Practical Exercise
- Step-by-step task

## Evaluation
- How to verify correctness

## Further Reading
- 3–6 links or references (no paywalled citations required)
