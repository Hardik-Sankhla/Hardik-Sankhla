# AI Project Case Study Prompt

Write a production-grade AI system case study.

## Tone
- Clear, technical, engineering-first
- No hype, no marketing fluff
- Use concrete details and measurable outcomes

## Audience
- Engineers, ML practitioners, technical hiring managers

## Output format
- Markdown
- Use the exact section headers below (H2)

## Required sections

## Problem
- Real-world context and constraints
- What “success” means (metrics, reliability, cost, latency)

## Approach
- Key design decisions and trade-offs
- Why this approach vs alternatives

## Architecture
- Components and data flow
- Interfaces and failure handling
- Observability plan (logs/metrics/traces)

## Tech Stack
- Languages, frameworks, services

## Results
- Quantitative outcomes where possible
- Reliability improvements and operational impact

## Learnings
- What changed your mind
- What you’d do differently next time
