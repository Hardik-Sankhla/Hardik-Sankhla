# Engineering Guide Prompt

Write a practical guide for building production AI systems.

## Tone
- Direct, actionable, technical
- Prefer checklists, patterns, and pitfalls

## Output format
- Markdown
- Use clear H2 headings

## Required structure

## Why This Matters
- When this shows up in real projects
- What breaks if ignored

## Core Concepts
- Definitions and mental models

## Implementation Checklist
- Concrete steps
- What to measure

## Common Failure Modes
- Symptoms and root causes

## Examples
- Short examples (pseudo-code is OK)

## Key Takeaway
- One paragraph summary
